Assumptions

  • Ideal transformer: perfect magnetic coupling, no winding resistance, leakage, core loss or magnetising current.
  • Sinusoidal AC steady state; voltages and currents are consistent RMS values.
  • Turns are positive and the core is not saturated.
  • An ideal transformer does not operate from steady DC; real output power is lower than input power.

How it works

The induced voltage per turn is the same in both windings, so voltage ratio equals turns ratio. More secondary turns step voltage up; fewer step it down.

With no losses, input and output powers are equal. Current therefore changes inversely with voltage so VₚIₚ = VₛIₛ.
