# Release Notes

The permanent changelog for Ans. The newest version is listed first.

## 1.0

*Initial release.*

- A full scientific calculator with the feel of a dedicated device: real
  keys, a two-line display, and instant response.
- Exact mathematics — fractions stay fractions, and π is kept exact, so
  ⅓ × 3 is exactly 1.
- Complete function set: trigonometry, logarithms, powers, roots of any
  degree, powers of ten and random numbers.
- Six variables with names, a variable editor, and independent memory keys.
- Replay earlier calculations and edit any expression in place.
- Hardware keyboard support on iPad.
- Home Screen and Lock Screen widgets keep the latest answer close; Home Screen
  widgets can page through recent completed calculations
  with calculator-style ▲ and ▼ keys without opening Ans.
- A Control Centre control opens Ans in one tap.
- No ads, no subscriptions, no tracking — and it works entirely offline.
